# Projects

## Vision for Pollution: Deep Learning to Detect Visual Pollution

**Type:** MSc Dissertation Project  
**Institution:** The University of Edinburgh  
**Period:** June 2025 - August 2025

This project focused on detecting and quantifying urban visual pollution using deep learning and computer vision.

Key work:

- Built a pixel-level segmentation dataset covering six urban visual pollutant types.
- Trained supervised segmentation models using PyTorch, including U-Net, DeepLabV3+, FPN, and SegFormer.
- Benchmarked open-vocabulary segmentation models, including CLIPSeg and OVSeg.
- Used prompt engineering and threshold optimization for open-vocabulary models.
- Used Optuna to tune hyperparameters for all models.
- Achieved 0.734 mIoU with the best supervised models.
- Conducted robustness testing under noise, blur, low contrast, and occlusion.
- Adapted the Visual Pollution Index to support segmentation masks for area-based scoring across multiple regions.

## Beyond GANs: A Comparative Study of Generative Models and Traditional Augmentation for Pothole Detection

**Type:** MSc Coursework  
**Institution:** The University of Edinburgh  
**Period:** January 2025 - March 2025

This project compared generative and traditional data augmentation methods for pothole detection.

Key work:

- Compared Diffusion Models, WGAN-GP, SinGAN, and Albumentations for augmenting the Road Damage Dataset 2022 pothole dataset.
- Generated synthetic potholes using GANs and Diffusion Models.
- Blended synthetic potholes into road images using Poisson blending.
- Trained YOLOv8n for cross-country generalization.
- Evaluated object detection performance using mAP, precision, recall, and F1-score.

## Stock Network Analysis with Graph Theory, MSFT Focus

**Type:** MSc Coursework  
**Institution:** The University of Edinburgh  
**Period:** September 2024 - October 2024

This project analyzed investor-stock relationships and stock correlations using graph theory.

Key work:

- Built investor-stock bipartite networks.
- Projected bipartite networks into stock-side networks to analyze co-investment patterns.
- Applied community detection and centrality metrics, including eigenvector centrality, betweenness centrality, and closeness centrality.
- Studied Microsoft's influence within network structures.
- Constructed a minimum spanning tree from stock correlations to examine sector clustering.
- Found that Microsoft was consistently central across network structures, especially within the technology cluster.

## Machine Learning Feasibility Study for Ship Valuation Automation

**Type:** Bank Mandiri Exploratory Machine Learning Project  
**Period:** September 2023 - October 2023

This proof-of-concept project explored whether machine learning could support collateral valuation automation for ships.

Key work:

- Conducted exploratory data analysis on vessel characteristics.
- Engineered features from ship dimensions, engine specifications, age, and capacity.
- Trained and compared predictive models, including Ridge Regression, Lasso Regression, Decision Tree, Random Forest, KNN, and XGBoost.
- Used a vessel valuation dataset for model experimentation.
- Created a Streamlit UI prototype to demonstrate interactive valuation predictions for non-technical stakeholders.
- Produced findings that informed the digital transformation roadmap for collateral valuation processes.
