### Dissertation

**Title:** Vision for Pollution: Deep Learning to Detect Visual Pollution

In this dissertation project, Yoseph built a pixel-level segmentation dataset and evaluated both supervised and open-vocabulary models for visual pollutant detection. The supervised models included U-Net, DeepLabV3+, FPN, and SegFormer. The open-vocabulary models included CLIPSeg and OVSeg.

The project focused on detecting and quantifying urban visual pollution using deep learning and computer vision.
